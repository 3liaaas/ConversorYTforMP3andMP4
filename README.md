# Conversor de Vídeos para arquivos MP4 e MP3

Esse programa foi pensado em um problema que já tive, que quando edita algum vídeo ou criava alguma música, tinha que baixar algum vídeo do Youtube por sites de terceiros, mas os sites eram irregulares e instáveis,
então decidi criar o meu próprio conversor para satisfazer as minhas necessidades

## E como o programa funciona?

- Ele vai pedir a URL(Youtube) para converter
- Irá perguntar em qual formato será feito o download, em MP3 ou MP4
- O programa será executado e irá mostrar o progresso
- Quando estiver pronto, o arquivo estará no local onde foi instalado o programa

## Ferramentas externas
O Programa necessita de duas ferramentas externas para ser rodado, que são:
- yt-dlp
- ffmpeg
